Cardano LSTM Models

Overview

The figures below depict the outcomes of long short-term memory neural networks using the features specified in each model’s figure caption to predict the future closing prices for a cryptocurrency. Each model takes seven consecutive days’ worth of information to predict the price one week in the future from the last day in the input data. Models were trained on the 75% of the data set (the earliest days) and then tested against the final 25% (the most recent days). Each graph shows the actual price value plotted against the predicted value. Dots on the red line (slope = 1) are cases where the prediction matched the real value. Dots above the line are cases where the actual price was higher than predicted. Dots below the line are cases where the actual price was lower than predicted.

Market Data

Topic

The actual and predicted dollar value for this cryptocurrency when using only market data from Yahoo Finance as the model features. The features included opening price, high price for the day, low price for the day, and volume traded.

Reddit Data

Topic

The actual and predicted dollar value for this cryptocurrency when using only post data from Reddit as the model features. The features included the number of posts, average number of comments per post, average score per post, and average compound sentiment score for post titles for the day. The compound sentiment score is a combination of how positive, negative, and neutral the post’s title was. A score of 1 is a fully positive title, while a score of 0 is a fully neutral title, and a score of -1 is a fully negative title.
